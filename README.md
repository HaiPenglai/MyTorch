## ⬇️ 如何下载

方法一：直接下载ZIP
1. 点击页面中绿色按钮 `Code`
2. 点击 `Download ZIP` 

方法二：使用Git克隆
```bash
git clone https://github.com/HaiPenglai/MyTorch.git
```

## 🛠️ 如何安装mytorch

下载完成后

```shell
cd MyTorch\solution
pip install -e mytorch
```

即可安装参考答案实现的mytorch，自己的mytorch需要在`MyTorch\exercise`中实现后同理安装


检查安装正确性:

```python
import mytorch
print(mytorch.__version__)
```
